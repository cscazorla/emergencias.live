# emergencias.live

Teléfonos de emergencia de España. Página ultra-ligera, sin JavaScript, optimizada para conexiones lentas.

## Características

- **Ultra-ligera**: < 5 KB
- **Sin JavaScript**: Funciona en cualquier navegador
- **Modo oscuro**: Automático según preferencias del sistema
- **Llamada directa**: Enlaces `tel:` para llamar con un toque
- **Accesible**: HTML semántico, enlaces grandes y táctiles
